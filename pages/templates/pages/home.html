Frequently Asked Questions

What is LLM Hub?

LLM Hub is a platform that allows you to chat with various Large Language Models (LLMs) in one place. It provides a unified interface to interact with different AI models, making it easy to explore and utilize their capabilities.

Do I need an API key to use LLM Hub?

Yes, you'll need to provide your own API key (starting with OpenAI) to use LLM Hub. This ensures that you have control over your usage and billing. You can easily add your API key in your profile settings after registering.

Which LLM models are supported?

We currently support OpenAI's models, including GPT-3.5 and GPT-4. We're continuously working on adding support for more models from different providers to give you a wide range of options.

Is my data secure?

Yes, we take data security very seriously. Your conversations and API keys are encrypted and stored securely. We do not store or use your data for any purpose other than providing the service to you.
